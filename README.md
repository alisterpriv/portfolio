## ✨ Overview

This portfolio was designed to present my work, technical skills, and projects in a clean and modern interface.

The design process started in **Figma**, where I explored modern UI layouts and took inspiration from **Figma Make** and **Google Stitch**.  
After finalizing the design, I implemented the website from scratch using modern front-end technologies.

The site includes:

- A modern hero section
- Skills and experience overview
- Featured projects
- Contact form with email functionality
- Fully responsive design
